# 会员 / 订阅口径与护栏

这个文件定义会员 / 订阅专项库默认遵守的口径和护栏。

## 口径规则

### 1. 试用转付费不等于续费

以下指标必须分开看：

- 试用转付费率
- 首购后首次续费率
- 已付费用户续费率
- 到期召回率
- 支付失败恢复率

不要把它们加总后称为一个“续费率”。

### 2. 生命周期状态必须单列

至少拆成：

- 在期未解约
- 在期已解约
- 已过期短窗
- 已过期长窗
- 支付失败待恢复

### 3. 最低分层要求

至少做以下三维交叉中的两层：

- 地区 / 国家
- SKU / 周期 / 套餐
- 生命周期状态

如有条件，再加：

- 渠道来源
- 新老用户
- 高价值 / 低价值用户

## 护栏指标

会员 / 订阅策略除了主指标，还应同时看：

- 投诉率
- 退款率
- 退订率
- 优惠渗透率
- 次周期续费表现
- 毛利变化

## 默认时间窗口

- 临期：T-7 / T-3 / T-1
- 过期：D1 / D7 / D14 / D30
- 新增早期：D0 / D3 / D7

## 实验边界

- 涉及价格和权益动作，优先灰度
- 同一批用户不要同时叠加多个核心变量
- 没有对照组时，结论默认只能说相关性
