## 使用的会员订阅专项 skill

| skill | 用途 | 本文中的使用方式 |
| --- | --- | --- |
| `subscription-renewal-diagnostic` | 诊断续费、解约、到期回流、国家 / SKU / 生命周期状态下的续费问题 | 作为总诊断主框架，用于拆市场、拆问题、拆优先级 |
| `cancellation-save-strategy` | 设计取消订阅拦截、降档、优惠挽回、暂停订阅和退出问卷策略 | 用于判断哪些市场适合做取消挽回、降档、年卡权益拦截 |
| `expired-member-winback-playbook` | 设计过期会员的短过期 / 长过期召回节奏、权益组合和实验方式 | 用于判断哪些市场更应做过期召回而非首购策略 |
| `payment-failure-recovery-playbook` | 设计自动续费失败、支付方式失效和非自愿流失恢复链路 | 用于东南亚市场的支付失败、扣款补救和本地支付恢复动作 |
